What is this?

This is a software viewer of a physical puzzle (Smartgames IQ mini hexpert) where you have to put 6 different shapes into a hexagonal grid. There are also 3 "stops" that move along a track that is used to set up different puzzles.

What makes this interesting is that wherever you put the stops, it's always possible to fit the pieces!

This toy has 2 modes: Setup and Solution. In Setup mode you can see the track and use the ABC sliders to move the stops along the track. In Solution mode it just show the solution, if I've solved it.

There are 2 special spots where it's not possible to put the stops, those are shown in grey in Setup mode

In total there are 9 x 9 x 7 = 567 puzzles to solve! Have fun!
